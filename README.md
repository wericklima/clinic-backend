# 🏥 Clinic Backend API

API backend para gerenciamento de pacientes de uma clínica, desenvolvida com **NestJS** e **MongoDB**.

---

## 🚀 Tecnologias utilizadas

* Node.js
* NestJS
* MongoDB Atlas
* Mongoose
* TypeScript

---

## 📌 Funcionalidades

* Criar paciente
* Listar pacientes
* Estrutura preparada para CRUD completo

---

## ⚙️ Como rodar o projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/wericklima/clinic-backend.git
```

### 2. Instalar dependências

```bash
npm install
```

### 3. Criar arquivo `.env`

```env
MONGO_URI=sua_string_do_mongodb
```

### 4. Rodar o projeto

```bash
npm run start:dev
```

---

## 📡 Endpoints

### Criar paciente

POST /patients

### Listar pacientes

GET /patients
